# GI — ESP32 Controller

Автоматизированная система управления гроубоксом на базе ESP32: климат-контроль, освещение, вентиляция, автополив и обогрев с веб-интерфейсом управления и облачной телеметрией.

## Возможности

- **Климат-контроль** — датчик SHT4x (температура/влажность), поддержание заданной температуры с гистерезисом
- **Освещение** — управление LED по расписанию день/ночь, яркость связана с температурным контролем
- **Вентиляция** — два независимых канала PWM-управления вентиляторами (fan1/fan2) с настраиваемыми мин/макс пределами
- **Автополив** — планировщик по дням недели (битовая маска), защита от повторного срабатывания
- **Обогрев** — реле обогревателя с гистерезисом, три режима расписания: день/ночь/всегда
- **Безопасность** — аварийное отключение при потере связи с датчиком SHT4x, защита от деления на ноль в расчётах
- **Веб-интерфейс** — вкладки настроек, интерактивные SVG-графики с тултипами, обслуживается через SPIFFS/LittleFS
- **Хранение данных** — логирование в CSV на SD-карту, настройки — в NVS (Preferences)
- **Облако** — телеметрия в Ubidots через отдельную FreeRTOS-задачу на Core 0

## Технологический стек

- ESP32 (Arduino framework)
- AsyncWebServer
- SPIFFS / LittleFS
- RTClib (планирование день/ночь)
- Adafruit SHT4x
- SD-карта (CSV-логирование)
- Ubidots (облачная телеметрия)

## Примечание по безопасности

Учётные данные WiFi и Ubidots хранятся в NVS Preferences и редактируются через веб-интерфейс. Значения по умолчанию в коде используются только при первой прошивке.
